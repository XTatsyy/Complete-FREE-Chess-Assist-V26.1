             <h1>Introducing Chess Assist V26.1</h1>
</div>

A Java based Chess Assist Extension which helps you progress and get better at chess faster. The solution to fastest chess progress is here! The chess assist provides you real time moves, analysis, eval bar for completely free! 

Our project's main goal is to provide better chess growth facilities to people who are bad at chess and cannot afford chess coaching or any other chess training material. The project is completely open-source and free.

## Features

* 5 Different Assist Versions (Machine, Human, Neural, Style, Fusion)
* Stockfish 19, Human like modification of Stockfish.
* Auto-Mode
* 8 different levels of each mode
* On-going game detection
* Free Analysis For Best Move
* Move Feedback


<div align="left">
            <h2>How to Install?</h2>
</div>

* Download the zip file from the repository
* Extract the zip file
* Go to chrome://extensions or click manage extensions
* Enable developer mode (Important)
* Click load unpacked, at the top left corner.
* Select the folder (Not the contents of the folder!) and Open it.
* Now in all extensions, turn on the Chess Assists V26 Extension
* Go to chess.com or lichess.
* Pin the extension and click it.
* Sign up with random username and password. 
* Now you're all ready to use the extension! 

Enjoy the extension and do not use it for *cheating*. Cheating will cause eventual ban.
